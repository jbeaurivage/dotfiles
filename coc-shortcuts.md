# Coc neovim shortcut cheatsheet

## Auto-completion

* Next selection: Tab

* Previous selection: Shift-Tab

* Trigger selection: Ctrl+Space

## Code navigation

* Previous diagnostic: [g

* Next diagnostic: ]g

* Go to definition: gd

* Go to type definition: gy

* Go to implementation: gi

* Reference: gr

* Open documentation: K

## Refactoring

* Rename symbol: <leader>rn

* Format selected code: <leader>f

* Apply codeAction to selected region: <leader>a
Example: <leader>aap for current paragraph

* Apply codeAction to the current buffer: <leader>ac

* Apply AutoFix to problem on current line: <leader>qf

## Text objects

* Function: {i|a}f

* Class: {i|a}c

## Misc

* Scroll float windows/popups: Ctrl+f and Ctrl+b

* Selection Range: Ctrl+s

* Format current buffer: :Fmt

* Fold current buffer: Fold

* Organize imports of current buffer: :Org
